# lab1
재고 관리 시스템 실습 과제 파일입니다.

## 파일 구조

```text
content/
  server.js
  static/
    images/
    style.css
  templates/
    material-detail.html
    material-edit.html
    material-list.html
    material-message.html
    material-write.html
```

## 설명

- `server.js`: Express 서버와 MariaDB 연결 코드
- `static/`: CSS와 이미지 파일
- `templates/`: 화면 템플릿 파일
